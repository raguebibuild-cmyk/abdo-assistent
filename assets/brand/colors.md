# DEGISaaS Brand Colors

## Primary Palette

| Name | Hex | RGB | Use |
|------|-----|-----|-----|
| Navy | `#012240` | 1, 34, 64 | Headers, table headers, nav bars, primary backgrounds |
| Amber | `#e89820` | 232, 152, 32 | "SaaS" logotype colour, accent dividers, highlights, CTAs |
| Indigo | `#4f4cee` | 79, 76, 238 | "DEGI" logotype colour, H3 headings, labels, links |

## Supporting Palette

| Name | Hex | Use |
|------|-----|-----|
| Body text | `#1e293b` | All body copy |
| Muted | `#64748b` | Footers, captions, metadata |
| Border | `#e2e8f0` | Table borders, subtle dividers |
| Alt row | `#f8fafc` | Table alternate row background |
| Code bg | `#f1f5f9` | Code block background |
| Code text | `#374151` | Code block text |
| Blockquote bg | `#fff8ed` | Blockquote background (warm amber tint) |

## Logo Usage

- File: `assets/brand/logo.png`
- Transparent background version preferred for placement on dark backgrounds
- Minimum clear space: equal to the height of the "D" in DEGI on all sides
- Do not recolour, stretch, or modify the logo

## Tagline

**Built to Scale**

Used beneath the logo. Set in navy `#012240`.
