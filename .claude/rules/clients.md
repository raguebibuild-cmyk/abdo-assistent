# Client Handling Rules

## Client Lifecycle
Every client moves through this arc: Prospect → Onboarding → Active → Invoice → Closed.
Track where each client sits before taking any action.

## Communication
- All client emails: draft only — never send without review
- Tone: professional but warm (see voice.md)
- Response goal: one clear ask or next step per message

## Quotes and Invoices
- Never send a quote or invoice without Abderrahim reviewing it first
- Use the invoice template in templates/ when built
- Flag if pricing hasn't been confirmed for a new service

## Audits
- Customise per client — never send a generic audit
- Use the audit template in templates/ when built
